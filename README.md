## Project Purpose

The purpose of this project is to create a visually rich and emotionally warm birthday website for a mentor. Instead of a simple birthday card, this project presents the wishes in the form of an animated one-page web experience.

It highlights appreciation for a mentor who teaches beyond code, inspires students, and helps them grow with confidence. The website is meant to feel premium, respectful, celebratory, and personal. :contentReference[oaicite:2]{index=2}
